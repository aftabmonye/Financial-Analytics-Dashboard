# 📊 Financial Analytics Dashboard

## Overview

This project demonstrates an end-to-end Financial Analytics solution using SQL, Python, and Power BI. The objective was to analyze financial performance, identify business trends, and build an interactive dashboard for decision-making.

---

## Tools & Technologies

* Power BI
* SQL
* Python
* Pandas
* NumPy
* Matplotlib
* Microsoft Excel

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. SQL Business Queries
5. Data Modeling
6. DAX Measure Creation
7. Dashboard Development
8. Business Insights

---

## Dashboard Features

* Total Revenue
* Total Expense
* Total Profit
* Profit Margin %
* Units Sold
* Monthly Revenue & Profit Trend
* Revenue by Product
* Revenue by Customer Segment
* Profit by Region
* Budget vs Actual Revenue
* Interactive Filters
